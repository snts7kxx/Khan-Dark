# 🎓KhanDark

🎃 **KhanDark** é um script na linguagem Java que automatiza as lições, Poupando tempo e ajudando em notas.

# ❓️ Como funciona?

> ✅️ Faz suas lições corretamente.

> 🔎 Usado como BookMarklet

> 🎬 Assiste videos, Concluindo rapidamente.
